# Introduction to LDA Topic Modelling
This repository is for the Silent Disco: Introduction to LDA Topic Modelling workshop

The tutorial is developed for CDCS by Aybuke Atalay and Somya Iqbal. This workshop uses the programming language Python and R to perform topic modelling using the Latent Dirichlet Allocation (LDA) method. The aim is that by using the pre-supplied code examples, and editing them to examine the different results, you will begin to gain an understand of the techniques used.

This repository contains the materials for the Silent Disco: Introduction to LDA Topic Modelling workshop.

The tutorial was developed for the Centre for Data, Culture & Society (CDCS) by Aybuke Atalay and Somya Iqbal.

In this workshop, participants will learn how to perform topic modelling using the Latent Dirichlet Allocation (LDA) method. The materials are provided in both Python and R, allowing participants to follow the version that best matches their programming experience.

Using the supplied scripts and dataset, participants will walk through the main steps of topic modelling: text preprocessing, fitting an LDA model, evaluating model choices, and interpreting topic outputs.

##  Workshop Aims
By the end of this workshop, participants should be able to:
- Understand the basic principles behind LDA topic modelling
- Perform text preprocessing for topic modelling
- Implement an LDA model on a text corpus
- Interpret the topics generated by the model
- Evaluate modelling choices, including selecting an appropriate number of topics
- Visualise topic modelling outputs
